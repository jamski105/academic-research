## Repo-Conventions
- Branch: feat/{milestone}-{chunk_id}
- Tests: pytest (`pytest tests/test_<...>.py`)
- Tickets im Schema: `## Scope`, `## Akzeptanzkriterien`, `## Out of Scope`
- Squash-Merge auf main via PR
- Plugin-Design: Agenten/Skills vor Python-Skripten bevorzugen (siehe memory `feedback_plugin_design`)
- Excel: document-skills:xlsx Skill, nicht openpyxl direkt (siehe memory `feedback_excel_skill`)
- Browser-Automation: browser-use Skill, nicht Playwright (siehe memory `feedback_browser_use`)
